ModelFind(model, what, callback)

Instead of Model.find(), uses Model.aggregate() to look for documents Supposedly "learns" from what it does...
Parameters:
Name Type Description
model Schema the model to find the documents in; ex: User, ForumSetting

  • models/index.js, line 108
